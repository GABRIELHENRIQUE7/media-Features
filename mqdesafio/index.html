Usando Media Queries

Uma consulta de mídia consiste em um tipo de mídia e pelo menos uma expressão que limita o escopo das folhas de estilo usando recursos de mídia , como largura, altura e cor. Media queries , adicionadas no CSS3 , deixam a apresentação do conteúdo adaptada a uma gama especifica de dispositivos não precisando mudar o conteúdo em si.

Sintaxe

Media queries consistem de um media type e podem, a partir de uma especificação CSS3, contendo uma ou mais expressões, expressas em media features , que determinam ou verdadeiro ou falso. Os resultados da consulta são verdadeiros se o tipo de mídia especificado na consulta de mídia corresponder ao tipo do documento exibido no dispositivo e todas as expressões na consulta de mídia são verdadeiras.

Quando uma media query é verdadeira, a camada de estilo ou as regras de estilos correspondentes são aplicadas, seguindo o padrão de regras de cascatas. Camadas de estilos com media queries funcionam a tag link vão fazer download mesmo se suas medias queries retornarem falso (eles não se aplicam, no entanto). A menos que você use os operadores notou only, o tipo de mídia é opcional e o tipo allserá implícito.
